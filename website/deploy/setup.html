Preparando o seu Computador

Hardware e Software

É essencial que cada aluno tenho seu próprio laptop ou desktop durante o curso. O curso foi desenhado para ser independente to sistema operacional, mas é altamente recomendado o uso de de um sistema Unix-based (como Linux).

Instalando Python

Nesse curso utilizaremos Python 3, mas a sintaxe que utilizaremos será aproximadamente 99% compatível com Python 2.7.

Módulos

Além do interpretador Python nos utilizaremos uma gama de pacotes científicos:

  • NumPy: Módulo fundamental para computação científica em Python.
  • SciPy: Ecossistema em Python baseado em software livre para matemática, ciência, e engenharia.
  • matplotlib: Biblioteca de plotagem 2D.
  • IPython: Ambiente Python interativo

Instalação

Para instalar tudo eu recomendo o instalador Anaconda, um produto gratuito oferecido pela ContinuumIO.

  1. Baixe e instale Anaconda no seu sistema utilizando o link acima. 
    [~]$ bash Anaconda-1.x.x-Linux-x86.sh
    Substitua Anaconda-1.x.x-Linux-x86.sh pela versão que você baixou.
  2. Abra uma janela do terminal, e acerte o seu $PATH para apontar para a instalação do Anaconda. Test com 
    [~]$ which python
    O resultado deve mostra o caminho da instalação nova.
  3. Atualize a sua distribuição Anaconda digitando 
    [~]$ conda update conda
    (Conda é o gerenciador de pacotes do Anaconda.)
  4. Adicione um canal de módulos 
    [~]$ conda config --add channels https://conda.binstar.org/ocefpaf
    (Conda é o gerenciador de pacotes do Anaconda.)
  5. Adicione um canal de módulos 
    [~]$ conda config --add channels https://conda.binstar.org/ocefpaf
    (Conda é o gerenciador de pacotes do Anaconda.)
  6. Instale o software necessário para o curso com 
    [~]$ conda create -n curso --file req.txt
    (Baixe aqui o arquivo req.txt.)
  7. Confira se o seu IPython notebook está funcionando corretamente: digite 
    [~] ipython notebook
    e uma janela do seu browser deve abrir mostrando o notebook.

Instalando Git

Git pode ser facilmente instalado no linux. Instruções mais detalhadas podem ser encontradas aqui.

Linux

use o gerenciador de pacotes da sua distribuição linux 

[~]$ sudo zypper in git # OpenSUSE
ou 
[~]$ sudo apt-get install git # Debian/Ubuntu
ou 
[~]$ yum install git-core # Fedora/RedHat

Windows

Git pode ser instalado em windows usando as ferramentas disponíveis em http://msysgit.github.com/.

Baixando o Material do Curso

Depois de instalar Git você pode baixar todo o conteúdo do curso (incluindo o código fonte HTML dessa página) digitando 

[~]$ git clone https://github.com/ocefpaf/python4oceanographers_intro_course.git
Caso queira se manter atualizado com o conteúdo entre na pasta criado e digite 
[~]$ cd python4oceanographers_intro_course
[~]$ git pull
